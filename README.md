# SwiftSearch

SwiftSearch は、指定したディレクトリ以下を高速に再帰検索し、  
ファイル名にキーワードを含むファイルを一覧表示する C# コンソールツールです。

Directory.EnumerateFiles を使用することで、  
全件読み込みを待たずに検索を進められるため、大規模フォルダでも高速に動作します。

---

## 特徴

- キーワードを含むファイルを高速検索  
- サブフォルダを含めた再帰検索  
- EnumerateFiles によるストリーミング検索で高速  
- 検索時間をミリ秒表示  
- 結果をそのままコピーして利用可能  
- アクセス権限エラーを自動スキップ  
- 完全ローカル動作、インストール不要

---

## 使い方

1. プログラムを実行  
2. 検索したいディレクトリパスを入力  
3. ファイル名に含めたいキーワードを入力  
4. ヒットしたファイル一覧が表示されるのでコピーして利用できます

---

## 想定される用途

- 大量のプロジェクトフォルダから特定ファイルを探す  
- バックアップフォルダ内のファイル名検索  
- 拡張子や部分一致での高速ファイル探索  
- Windows 標準検索より高速に結果を得たい場合
